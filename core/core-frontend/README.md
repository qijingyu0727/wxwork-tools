# 企业微信工具前端

企业微信智能表格管理系统的前端项目，基于 Vue3 + Vite 构建。

## 技术栈

- Vue 3
- Vite
- Vue Router
- Pinia
- Tailwind CSS
- Axios

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 项目结构

```
src/
├── api/          # API 请求封装
├── components/   # 公共组件
├── router/       # 路由配置
├── stores/       # Pinia 状态管理
├── styles/       # 全局样式
├── utils/        # 工具函数
├── views/        # 页面组件
├── App.vue       # 根组件
└── main.js       # 入口文件
```
